# YourProject_Team13_Map
Карта литературных топонимов - интерактивное веб-приложение для визуализации географических мест из биографий писателей и литературных произведений. Проект объединяет цифровые гуманитарные науки, геоинформационные технологии и образовательные задачи. Проект создан в рамках конкурса "Твой проект".
