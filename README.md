# Карта литературных топонимов

Интерактивное веб-приложение для визуализации географических мест из биографий писателей и литературных произведений. Прототип реализован на примере Н.В. Гоголя.

## Функциональность

- Два режима просмотра: биография писателя и литературное произведение
- Фильтрация топонимов по типу (реальные / вымышленные)
- Поиск по названиям топонимов
- Интерактивная карта с маркерами
- При клике на маркер отображается контекст события или цитата из произведения
- В биографическом режиме отображается хронологический маршрут перемещений

## Структура данных

Прототип включает данные по Н.В. Гоголю:
- Биография: 13 топонимов
- Повесть «Шинель»: 7 топонимов
- Повесть «Тарас Бульба»: более 70 топонимов

Каждый топоним содержит:
- название
- координаты
- тип (реальный / вымышленный / биографический)
- контекст (предложение из текста)
- год события (при наличии)
- частоту упоминаний

## Технические требования

Для корректного отображения необходимо:

- Современный веб-браузер с поддержкой JavaScript (Chrome, Firefox, Safari, Edge последних версий)
- Активное интернет-соединение для загрузки картографических тайлов (OpenStreetMap)
- Разрешение экрана не менее 1024x768 для комфортного отображения интерфейса

## Запуск

1. Скачайте файл `index.html`
2. Откройте его в любом современном браузере двойным кликом или через меню «Открыть файл»

Дополнительный веб-сервер не требуется.

## Используемые технологии

- Leaflet.js — картографический движок
- OpenStreetMap (CartoDB) — картографические тайлы
- HTML/CSS/JavaScript (без внешних зависимостей, кроме Leaflet)
